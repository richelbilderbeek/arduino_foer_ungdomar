# Lektion 35: Anslutning av en kapacitiv knapp

En kapacitiv knapp är en inte en vanligt knapp,
men en koppling för att trycka på avstånd (!).

Under den här lektionen ansluter vi en.

Men för att vi kan inte göra något utan en Arduino
med det, är vi färdiga :-)
